# Enterprise Data Trust Score Platform

## SQL Folder

This folder contains SQL scripts used in the project.

### Files

### 01_create_table.sql
Creates the Employees table.

### 02_insert_data.sql
Employee records are inserted using the Python ETL Pipeline.

### 03_analysis_queries.sql
Contains analytical SQL queries for:

- Complete Employee Data
- Total Employees
- Department Analysis
- Salary Analysis
- Trust Score Analysis
- Trust Category Analysis
- City Analysis
- Top Paid Employees
