# Enterprise Data Trust Score Platform

## 📌 Project Overview
Enterprise Data Trust Score Platform is an end-to-end Data Engineering and Data Analytics project developed using Python, SQL, SQLite, Pandas, and Power BI.

The project validates enterprise data quality, calculates trust scores, stores processed data in SQLite, and visualizes insights through an interactive Power BI dashboard.
